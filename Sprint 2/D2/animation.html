What are CSS Animations?

CSS animations make elements move, change or transform over time.

Why we use CSS Animations?

  • Make websites feel alive and interactive
  • Guide users attention to important elements
  • Provide visual feedback (hover buttons, loadings)
  • Create delightful user experiences
  • Yo do not need JavaScript for basic animations

Types of CSS Animations

  • Transforms Change shape, position, or size instantly. rotate, scale move, skew
  • Transitions smooth change from one state to another. hover effects and color changes.
  • Keyframe Animations complex, multi-step animations. Loading spinners, bouncing elements, slideshows, aniated menus and background.
