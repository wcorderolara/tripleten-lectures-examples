ABSOLUTE Positioning

Also removes an element fro the document natural flow and positions it relative to its nearest positioned acestor (any parent with position other than static). If no positioned ancestor exists, it positions relative to the document.

Key Characteristics

  • Removed from document flow
  • Positionede relative to neares ancestor
  • Takes up no space in the layout
  • Always must exists a parent and a child element
